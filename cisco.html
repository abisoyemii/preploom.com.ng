CCNA 200-301 Practice Questions (200+ Available)

Network Fundamentals Medium

What is the default administrative distance for a static route on a Cisco router?

  • A) 0
  • B) 1
  • C) 10
  • D) 110
Correct: B) 1
Static routes have AD of 1 (connected = 0, EIGRP = 90, OSPF = 110).
Network Access Easy

Which layer of the OSI model is responsible for logical addressing?

  • A) Layer 1
  • B) Layer 2
  • C) Layer 3
  • D) Layer 4
Correct: C) Layer 3
Layer 3 (Network) handles IP addressing.
IP Connectivity Medium

Which OSPF area type requires a default route from the ABR?

  • A) Area 0
  • B) Stub area
  • C) Totally stubby
  • D) NSSA
Correct: B, C, D
All non-backbone areas get default route from ABR.
Security Easy

What command configures SSH v2 on Cisco IOS?

  • A) crypto key generate rsa
  • B) ip ssh version 2
  • C) ssh version 2
  • D) transport input ssh
Correct: B) ip ssh version 2
Plus rsa key and vty transport input ssh.
Automation Hard

RESTCONF uses which HTTP methods for CRUD operations?

  • A) GET, POST, PUT, DELETE
  • B) GET, POST, PATCH, DELETE
  • C) GET, PUT, POST, DELETE
  • D) HEAD, GET, POST, PUT
Correct: A) GET, POST, PUT, DELETE
Standard RESTCONF CRUD methods.

ENCOR Hard

What Cisco command enables NETCONF on a router?

  • A) netconf ssh
  • B) netconf-yang
  • C) conf t netconf
  • D) feature netconf
Correct: D) feature netconf
Enable NETCONF feature on NX-OS/IOS-XE.
ENARSI Hard

Which command configures BGP route reflector?

  • A) neighbor rr-client
  • B) neighbor route-reflector-client
  • C) bgp rr-server
  • D) reflector client
Correct: B) neighbor route-reflector-client
BGP route reflector configuration.
