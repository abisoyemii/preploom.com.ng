Free Practice Questions

Data Analytics Medium

In data analysis, what does SQL primarily do?

  • A) Create data visualizations
  • B) Query and manipulate databases
  • C) Build machine learning models
  • D) Design user interfaces
Answer: B
SQL (Structured Query Language) is used to communicate with databases for retrieving, updating, and managing data.
UX Design Hard

Which UX research method involves observing users in their natural environment?

  • A) Surveys
  • B) A/B Testing
  • C) Field Studies
  • D) Card Sorting
Answer: C
Field studies observe real user behavior in natural context, revealing unfiltered insights.
