### 4. STRUCTURED DATA ENHANCEMENTS

#### Add WebSite Schema (Global - add to all pages)
```json
{
  "@context": "https://schema.org",
  "@type": "WebSite",
  "name": "PrepLoom",
  "url": "https://preploom.com.ng",
  "potentialAction": {
    "@type": "SearchAction",
    "target": "https://preploom.com.ng/search?q={search_term_string}",
    "query-input": "required name=search_term_string"
  }
}
```

#### Add Course Schema to Each Exam Page
Already present but ensure ALL exam pages have it with proper structure.

#### Add HowTo Schema for Study Guides
